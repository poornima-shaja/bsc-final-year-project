# The Groove Connection  
### BSc IT Final Year Project  

## 🚀 Project Summary  
The Groove Connection is a full-stack web application developed as part of my BSc IT final year project. The system integrates a deep learning model with a web interface to provide intelligent, data-driven functionality.

## 🛠 Tech Stack  
- Python  
- Flask  
- MySQL (XAMPP Server)  
- HTML5, CSS3, JavaScript  
- Convolutional Neural Network (CNN)  
- Trained Model (.h5 file)

## ✨ Key Features  
- Developed a Flask-based web application with structured backend architecture  
- Integrated MySQL database for efficient data storage and retrieval  
- Designed and trained a CNN model for prediction/classification tasks  
- Integrated trained `.h5` model with Flask backend for real-time inference  
- Implemented modular code structure and organized project architecture  

## 💡 Skills Demonstrated  
- Full-stack web development  
- Database design and integration  
- Deep Learning using CNN  
- Machine Learning model integration in web applications  
- Backend–ML pipeline connectivity
